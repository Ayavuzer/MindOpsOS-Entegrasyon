## UI Features

### Emails Page

- Table with columns: Subject, Sender, Type, Status, Received, Voucher
- Status filter: All, Pending, Processed, Failed
- Type filter: All Types, Booking, Stopsale
- PDF badge for emails with attachments
- Empty state with helpful message

### Reservations Page

- Card grid layout (3 columns on desktop)
- Each card shows: Voucher#, Hotel, Dates, Nights, Room, Guests, Price
- Status badge with sync indicator
- Currency-formatted prices

### Common

- AuthContext integration
- Tenant name in header subtitle
- Warning for unauthenticated users
- Refresh button on both pages
